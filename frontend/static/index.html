1H Forecast (predicts next hour)
BULLISH = close > MA20 AND MA20 > MA50
         AND ret_3bar > 0

BEARISH = close < MA20 AND MA20 < MA50
         AND ret_3bar < 0

else = NEUTRAL (no trade)
5-min Signal (entry trigger)
if forecast == BULLISH → BUY at close
if forecast == BEARISH → SELL at close
if forecast == NEUTRAL  → no trade

Stop    = entry ± 1.0 × ATR(20)
Target  = entry ± 2.0 × ATR(20)     # 1:2 R:R
Hold    = 12 bars max (60 min)

Session filter:
  skip Friday after 20:00 UTC
  skip Monday before 01:00 UTC
Risk (HARD < 20% Max DD)
risk_per_trade     = 0.1% of capital
max_dd_threshold   = 20%  HARD
daily_loss_limit   = 5%
max_leverage       = 0.5x (max notional/capital)
position size      = (cap × risk) / |entry − stop|
                     capped at max_leverage

if DD > 20%        → PAUSE (recover-pause)
if day_loss > 5%   → PAUSE for the day
if DD < 16%        → UNPAUSE (recovered)

⇒ 1 trade cannot lose more than 0.1% of capital.

  ↓  forecast → signal → submit to IG → record opportunity
SignalsService      (backend/core/signals_service.py)
  ↓  every signal + every opportunity (full audit trail)
RiskManager         (backend/risk/manager.py)

  ↓  20+ REST endpoints
